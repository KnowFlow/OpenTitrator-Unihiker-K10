# K10 电位滴定仪路线图

[English](ROADMAP.md) · [项目说明](../README_CN.md) · [使用说明书](MANUAL_CN.md)

状态更新至 **2026-07-14**。项目已基本完成从固定 pH 滴定仪到可配置电位滴定平台的核心升级。下一阶段重点不是继续堆叠控制界面功能，而是完成真实化学样品验证和生产设备验收。

## 状态说明

- **已完成**：功能已实现并有自动化检查覆盖。
- **待验证**：功能已实现，但仍需代表性化学实验或带标签设备验收。
- **规划中**：尚未实现。

## 已完成的基础能力

- [x] 四种方法预设：pH 终点、mV 终点、EDTA 硬度和手动方法。
- [x] 按方法保存 endpoint、target、信号趋势、滴定剂、样品/最大用量、EP 控制参数、结果公式、空白值、密度、手动浓度和手动系数。
- [x] 可配置控制区、终点保持、信号稳定阈值、静置上下限和最长运行时间。
- [x] 自适应脉冲加液、预测停泵、计时溢出保护、看门狗/安全停泵和校准有效性检查。
- [x] 可在主机测试的 `RunEngine`，统一管理暂停/恢复预热、加液、静置、终点保持、EQP、完成和紧急停止等实验生命周期。
- [x] 酸碱浓度、以 CaCO3 计的 EDTA 硬度和手动系数结果计算。

## 网页、记录与安全

- [x] pH/mV 实时曲线、用量/时间横轴、自动/手动 EQP、CSV/JSON 导出和仅供参考且不自动应用的参数建议。
- [x] 完整实验记录：样品编号、批次/参考号、操作员、备注、设备快照、结果、EQP 信息和可打印报告。
- [x] 浏览器 IndexedDB 仅保存最近 50 条已完成或中止记录；运行中记录只保存在内存。
- [x] 对当前曲线或导入的 Run Record JSON 做离线回放分析，包括合并重复用量点、三点局部斜率和质量提示。
- [x] 完整中英文网页，记忆语言选择，中文资源按需加载。
- [x] 所有写操作仅接受带认证 POST；统一命令/状态准入、限流、30 分钟无成功写操作会话过期、公开只读遥测和无需登录但只能停泵的紧急停止。
- [x] 每台设备独立的出厂恢复、USB 管理员恢复，以及带泵安全锁的认证 HTTP OTA。

## 当前验证关卡

- [ ] 使用覆盖预期浓度范围的真实硬度样品，调校 EDTA EQP 的最小峰值、回落比例和确认点数。
- [ ] 对比浏览器回放 EQP、固件停泵 EQP 和参考方法，定义可接受的偏差与重复性。
- [ ] 使用实际软管、滴定剂粘度、搅拌条件、电极响应时间和样品基质验证泵校准与结果准确度。
- [ ] 每台生产设备生成并粘贴唯一出厂凭据/标签，现场验证登录、恢复、停泵、OTA 失败、传感器故障和代表性实验。
- [ ] 为每台仪器记录已验证固件版本、校准数据和验收结果。

## 后续功能里程碑

### M7 — 钙/镁硬度流程

- [ ] 定义总硬度与单独钙硬度的实验顺序，包括 pH 10 和 pH 12–13 条件控制说明。
- [ ] 定义两次结果计算镁硬度的方法，以及空白值、密度、单位和不确定度的报告方式。
- [ ] 先在不增加第二支 pH 探头的前提下验证流程；只有实验表明确有必要时再扩展硬件。

### M8 — 可重复的实验室工作流

- [ ] 方法流程编辑器：组合取样、条件调节/混合、滴定、清洗、计算和报告步骤。
- [ ] 系列样品和批次队列，包含操作员和参考信息。
- [ ] 报告模板和多次实验对比，包括平行样统计和异常值复核。
- [ ] 带明确 schema 版本的浏览器记录备份/导入。

### M9 — 生产加固

- [ ] 带版本的设置/记录迁移和明确的降级策略。
- [ ] 评估泵电源硬件联锁、舱盖和试剂防护。
- [ ] 自动化生产与维护检查表，且任何密钥不得进入 Git 或日志。
- [ ] 发布产物、变更日志和可重复执行的生产冒烟测试包。

## 设计约束

- endpoint 信号、滴定剂、趋势和结果公式必须相互独立，不把引擎绑定到 NaOH/HCl。
- 高频曲线数据保存在浏览器，不写入 K10 flash。
- 回放、历史记录和报告功能不得发出任何设备控制命令。
- 每个里程碑都需要相应的本地/浏览器测试和固件构建；涉及化学或硬件行为时必须补充实机验证。
